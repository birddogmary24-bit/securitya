# 제품 요구사항 정의서 (PRD)
# AI 미국주식 브리핑 서비스

> **최종 업데이트:** 2026-03-22

## 1. 프로젝트 개요

**프로젝트명:** AI 미국주식 브리핑 서비스
**설명:** 한국 투자자의 미국 주식 정보 접근 격차를 AI/LLM으로 해소하는 서비스. 3가지 정보 풀(공식 데이터·공개 정보·집단 지성)을 구축하고, 투자자 페르소나 기반으로 선제적 인사이트를 제공.
**형태:** A사 앱 내 웹뷰(Add-on) 방식의 웹 서비스

## 2. 문제 정의

한국 투자자는 3가지 구조적 벽에 막혀 있다:

| 벽 | 설명 | 해법 (정보 풀) |
|----|------|---------------|
| **언어의 벽** | SEC 공시·어닝콜이 복잡한 금융 영어로 작성 | Pool 1 + Pool 2 (AI 수집·번역·분석) |
| **시차의 벽** | 미국장(한국 23:30~06:00)에서 발생하는 이벤트를 수면 중 놓침 | Pool 1 + Pool 2 (실시간 모니터링) |
| **맥락의 벽** | "EPS 상회"가 내 포트폴리오에 어떤 의미인지 해석해주는 서비스 없음 | 투자자 페르소나 + Pool 3 (집단 지성) |

## 3. 타겟 고객

| 세그먼트 | 프로필 | 핵심 Pain Point |
|---------|--------|----------------|
| **직장인 투자자** (30~40대) | 포트폴리오 $10K~$100K, 10~20종목 | 본업이 있어 모니터링 시간 부족. 밤사이 이벤트 대응 불가 |
| **성장형 초보투자자** (20~30대) | $1K~$10K, 1~2년차 | 커뮤니티/유튜브 의존, 허위정보 판별 불가 |

## 4. 솔루션 프레임워크

### 3가지 정보 풀
| 풀 | 정보 유형 | 매핑 기능 |
|----|----------|----------|
| **Pool 1: 공식 데이터** | SEC 공시, 실적발표, 경제 캘린더 | 기능 2 |
| **Pool 2: 공개 정보** | 뉴스, Reddit, X, 소비자 리뷰 | 기능 1, 3 |
| **Pool 3: 집단 지성** | AI 큐레이션 정보 기반 구조화된 토론 | 기능 3 확장 |

### 투자자 페르소나
- 온보딩 시 8가지 투자 특성을 1~5점으로 자가 평가
- 특성: 스윙매매, 장기투자, 스캘핑, 우량주, ETF, 소형주, 테크주, 배당주
- AI가 페르소나 기반으로 확증편향을 깨는 보완적 관점 제안

## 5. 핵심 기능

### 기능 1: AI 종목 브리핑 — 페르소나 기반 (MVP)
- 매일 오전 7시, 보유종목 + 투자자 페르소나 기반 개인화 브리핑 푸시
- Pool 1(공시·실적) + Pool 2(뉴스·소셜) 교차 분석 → 포트폴리오 관점 요약
- 종목 간 연쇄 영향 분석 (NVDA 가이던스 → AMD 영향)
- **페르소나 기반 선제적 제안**: 투자 특성의 사각지대를 비추는 보완적 관점
  - 예: "FOMC로 기술주 변동성 확대 가능. 현 포지션 유지하실 건가요?"
  - 예: "시장 상승기에 배당 외 성장 섹터 기회도 고려해보셨나요?"
- 개별 종목 매수/매도 추천 금지 (규제 가드레일)

### 기능 2: AI 공시 해석기
- 10-K/10-Q/8-K 자동 파싱 → 한국어 해석
- 핵심 지표 변동, 리스크 팩터 변화, 경영진 톤 분석
- 공시 제출 즉시 해석 전달
- 전분기/전년 대비 비교 자동 생성

### 기능 3: 구조화된 집단 지성 — AI 정보 기반 댓글·토론 (횡단 기능)
- AI를 통해 전달되는 **모든 정보**(브리핑 카드, 공시 해석, 뉴스 요약, 이벤트 알림)에 댓글 기능
- 별도 커뮤니티 탭이 아닌, 기능 1·2 전체에 걸쳐 적용
- AI 팩트체크: 댓글 내 사실 오류 자동 표시
- 기존 자유 게시판과 다름: AI 큐레이션 정보가 토론의 앵커 → 집중 토론
- 투자 조언 금지 가드레일

### 기능 4: AI 투자 맥락 Q&A 챗봇 (Phase 4 — TBD)
- 한국어 대화형 종목 분석 (공시/어닝콜/뉴스 기반), 출처 인용 필수

## 6. UX Flow

```
A사 앱
  └─ 해외주식 탭
       └─ "AI 투자비서" 배너
            └─ WebView 로드
                 ├─ [온보딩] 투자자 페르소나 설정 (8가지 특성 1~5점)
                 ├─ [홈] 오늘의 AI 브리핑 (페르소나 기반) + 댓글
                 ├─ [공시] AI 공시 해석기 + 댓글
                 ├─ [Q&A] AI 투자 맥락 Q&A 챗봇 (Phase 4)
                 └─ [설정] 관심종목/알림/페르소나 설정
```

## 7. 기술 아키텍처

```
[투자자 페르소나 엔진]
  8가지 특성 × 1~5점 → 개인화 기준 → 판단 엔진

[데이터 수집 — 3가지 정보 풀]
  Pool 1: SEC EDGAR / Finnhub / 경제캘린더
  Pool 2: NewsAPI / Reddit API / X API / 소비자 리뷰
  Pool 3: 구조화된 커뮤니티 (AI 큐레이션 게시글 + 사용자 댓글)
      ↓
[데이터 처리] 청킹 → 임베딩 → Vector DB (Pinecone/ChromaDB)
      ↓
[AI 분석] Claude API + RAG + 페르소나 판단 엔진
  → 포트폴리오 매칭 → 보완적 관점 생성 → 인사이트 생성
      ↓
[프론트엔드] Next.js + Tailwind (모바일 웹뷰 375px)
```
