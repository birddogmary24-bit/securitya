AI 미국주식 브리핑 서비스 — 과제 제출

안녕하세요.

서비스 개요: 한국 투자자의 미국 주식 정보 접근 격차를 AI/LLM으로 해소하는 서비스입니다. 3가지 정보 풀(공식 데이터·공개 정보·집단 지성)을 구축하고, 투자자 페르소나 기반으로 선제적 인사이트를 제공합니다.

형태: A사 앱 내 웹뷰(Add-on) 방식의 웹 서비스

MVP 데모 바로가기

주요 제안 내용

  • 문제 정의: 한국 투자자 47조원 미국주식 투자, 그러나 언어·시차·맥락의 3가지 구조적 벽으로 정보 비대칭 존재
  • 솔루션: 3가지 정보 풀(SEC 공시·뉴스/소셜·집단 지성) 구축 + 투자자 페르소나 기반 개인화 AI 브리핑
  • 핵심 차별점: 단순 번역·알림이 아닌, 사용자가 찾기 전에 AI가 먼저 맥락을 해석하고 선제적으로 인사이트를 제안
  • 기능 1 — AI 모닝 브리핑: 보유종목 + 페르소나 기반 개인화 브리핑. 확증편향을 깨는 보완적 관점 제공
  • 기능 2 — SEC 공시 해석기: 10-K/10-Q/8-K 자동 파싱·한국어 심층 분석 (한국 증권사 최초)
  • 기능 3 — 구조화된 집단 지성: AI 정보 기반 댓글·토론 시스템으로 커뮤니티 활성화
  • 검증 지표: North Star = AI 콘텐츠 열람 후 7일 내 거래 전환율. 브리핑 열람율 40%+, 페르소나 설정 완료율 80%+ 목표

MVP 서비스 핵심 기능 및 기술 사항

핵심 기능

  • 기능 포트폴리오 입력 + 8가지 투자 성향 페르소나 온보딩 → 개인화 AI 브리핑 생성
  • 기능 550종목 3-Tier 시스템: Tier 1(50) 전체 수집, Tier 2(100) 전체 수집, Tier 3(400) 기본+뉴스, On-demand 즉시 등록
  • 기능 SEC EDGAR 공시(10-K/10-Q/8-K) 자동 수집 + AI 한국어 요약
  • 기능 종목 검색 시 미등록 종목도 Finnhub에서 즉시 가져와 자동 등록 (On-demand)

기술적 주요 사항

  • 데이터 Finnhub API(무료) — 주가·재무·추천·목표가·내부자거래·어닝 등 8개 테이블로 구조화 수집
  • 데이터 SEC EDGAR API — CIK 매핑 + 90일 공시 수집, 청크 방식 배치 처리
  • AI Gemini 2.5 Flash/Flash-Lite 4단계 모델 fallback 체인 — 안정성 확보 + 비용 최적화
  • AI 종목별 AI 분석 사전 생성(Cron) → 브리핑 요청 시 캐시 조합으로 즉시 응답 (LLM 호출 최소화)
  • AI data_freshness_key(SHA256) — 데이터 미변경 시 AI 재생성 스킵으로 API 비용 절감
  • 인프라 GitHub Actions Cron(06:30 KST) — Vercel Hobby 10초 timeout 우회, 60분 배치 실행
  • 인프라 Supabase PostgreSQL 15개 테이블, RLS 보안 적용. 전체 무료 티어 운영
  • 인프라 Next.js 16 + Tailwind CSS 4 + Vercel 배포 — 모바일 웹뷰(375px) 최적화

향후 구현 예정

  • AI SEC 공시 원문 RAG 파이프라인 — 벡터 DB 기반 공시 심층 Q&A (Phase 2)
  • 기능 소셜 센티먼트 분석 — Reddit·X 여론 수집 + AI 감성 분석 통합 (Phase 2)
  • 기능 AI 정보 기반 댓글·토론 시스템 — 구조화된 집단 지성 커뮤니티 (Phase 3)
  • 기능 대화형 AI 투자 Q&A 챗봇 — 공시·어닝콜 기반 맥락 답변 + 스트리밍 (Phase 4)
  • 인프라 사용자 인증(Supabase Auth) + 푸시 알림 — 실시간 이벤트 알림 체계 구축
