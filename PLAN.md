## Phase 1: 브리핑 MVP
**목표:** AI 종목 브리핑 핵심 기능 동작 확인

1. **프로젝트 셋업**
   - Next.js + Tailwind CSS 초기 설정
   - Supabase DB 연결
   - 환경 변수 구성

2. **포트폴리오 입력 UI**
   - 종목 + 수량 입력 화면 (모바일 웹뷰 375px 기준)
   - 로컬 상태 관리

3. **데이터 수집 파이프라인**
   - Finnhub/yfinance 주가 데이터
   - NewsAPI 뉴스 수집
   - 경제 캘린더 API (FOMC 등)

4. **LLM 브리핑 생성**
   - Claude API 연동
   - 포트폴리오 기반 개인화 프롬프트
   - 선제적 제안 생성 ("오늘 밤 FOMC — 변동성 대비...")

5. **브리핑 카드 UI**
   - 카드형 종목별 브리핑 리스트
   - A사 디자인 톤 (밝은 배경, 노란색 포인트)
   - 종목별 컬러 코딩 (🔴/🟡/🟢)

6. **배포**
   - Vercel 배포

## Phase 2: AI 공시 해석기
**목표:** SEC EDGAR 공시 자동 파싱 및 한국어 해석

7. **SEC EDGAR 데이터 수집**
   - 10-K / 10-Q / 8-K 원문 자동 수집

8. **공시 파싱 + RAG 파이프라인**
   - 문서 청킹 → 임베딩 → 벡터 DB 저장
   - 관련 섹션 검색 (MD&A, Risk Factors 등)

9. **공시 해석 UI**
   - 종목 선택 → 최신 공시 해석 목록
   - 공시 유형 필터 (10-K/Q/8-K)

10. **전분기 비교 기능**
    - 핵심 지표 YoY/QoQ 자동 비교
