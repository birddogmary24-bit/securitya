# 경쟁사 벤치마크 분석: 스톡나우 & 토스증권

> **최종 업데이트:** 2026-03-21
>
> AI 미국주식 브리핑 서비스 경쟁사 분석
> 분석 기준일: 2026년 3월

---

## 1. 분석 개요

### 목적
AI/LLM 기술을 활용한 미국 주식 정보 접근 개선 제안을 위해, 동일 문제를 풀고 있는 핵심 경쟁 서비스를 구조적으로 분석하고 **A증권사이 공략할 수 있는 빈 공간**을 도출한다.

### 분석 대상

| 서비스 | 유형 | 핵심 포지션 |
|--------|------|------------|
| **스톡나우** | AI 정보 플랫폼 (스타트업) | 미국주식 정보 접근성 문제에 특화된 순수 정보 서비스 |
| **토스증권** | 모바일 증권사 | 해외주식 점유율 1위, AI 기능을 빠르게 확장 중인 직접 경쟁자 |

## 2. 스톡나우 (StockNow) 분석

### 2-1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 운영사 | 터네이셔스(주) (공동대표 김정훈, 최준회) |
| 설립 | 2023년 7월 |
| 서비스 론칭 | 2024년 10월 |
| 플랫폼 | iOS / Android / Web (stocknow.ai) |
| MAU | 3만명 (론칭 4개월 시점) |
| 파트너십 | 교보증권 MOU (2025년 11월) |
| 비즈니스 모델 | 프리미엄 구독 (무료 + 프로 월 16,900원) |

### 2-2. 핵심 기능 맵

| 계층 | 기능 | 상세 |
|------|------|------|
| **실시간** | 어닝콜 실시간 통/번역 | 국내 최초, ~4,000개 미국 상장기업 커버. 2025년 4월 한국어 음성 통역 추가 |
| **실시간** | 실시간 속보 뉴스 | 기관용 전문 데이터 벤더 라이선스 기반, 개인투자자에게 기관급 정보 제공 |
| **실시간** | 실적 캘린더 | 미국 기업 실적 발표 일정을 한국 시간으로 자동 변환 |
| **분석** | AI 실적 리포트 요약 | Revenue, EPS, 가이던스, 긍/부정 요인 해설 |
| **분석** | 속보 AI 분석 | 속보의 시장 맥락과 의미를 AI가 해석 |
| **소비** | AI 모닝브리핑 | 전일 미국 증시 요약을 5~7분 팟캐스트로 제공 |
| **소비** | 와치리스트 | 관심 종목 주가, 실적 일정 대시보드 |

### 2-3. AI 기술 파이프라인

```
[영어 음성 (어닝콜)]
    → ASR (음성인식)
    → LLM (번역 + 요약 + 분석)
    → TTS (음성합성)
    → 한국어 음성 실시간 통역

[영어 텍스트 (뉴스/SEC 공시)]
    → LLM (번역 + 요약 + 분석)
    → 구조화된 한국어 콘텐츠

[AI 팟캐스트 생성]
    → 전일 시황 자동 분석
    → 모닝브리핑 오디오 생성
```

### 2-4. 비즈니스 모델

| 티어 | 가격 | 주요 차이 |
|------|------|-----------|
| 무료 | 0원 | 속보/리포트 **3시간 지연**, 어닝콜 통역 **월 60분** |
| 프로 | 월 16,900원 | 속보 즉시, 리포트 실시간, 통역 무제한 |

> 기관용 유사 서비스(월 ~30만원) 대비 약 1/20 가격으로 진입장벽을 낮춤

### 2-5. 강점 요약

- **페인포인트 정조준**: "시차 + 언어장벽"이라는 서학개미 2대 고충을 정확히 타겟
- **국내 최초 어닝콜 실시간 통번역**: 4,000개 기업 커버, 경쟁 해자(moat) 확보
- **기관급 정보의 민주화**: 높은 가격 장벽을 낮춰 개인투자자 접근 가능
- **멀티포맷 콘텐츠**: 텍스트 + 오디오(팟캐스트) 다양한 소비 채널
- **검증된 시장 수요**: 론칭 4개월 MAU 3만, 프로 회원 뉴스 사용률 120%↑

### 2-6. 약점 / 빈 공간

| # | 약점 | 설명 |
|---|------|------|
| 1 | **매매 기능 없음** | 순수 정보 플랫폼. 정보→판단→매매 여정이 끊김 |
| 2 | **개인화 부족** | 사용자 포트폴리오 기반 맞춤 정보 제공 불가 |
| 3 | **맥락 연결 부재** | 뉴스가 "내 보유종목에 미치는 영향"으로 연결되지 않음 |
| 4 | **대화형 AI 없음** | 자연어 질의("테슬라 어닝콜 핵심이 뭐야?") 불가 |
| 5 | **커뮤니티 없음** | 정보 소비 후 토론/공유 기능 부재 |
| 6 | **생태계 연동 불가** | 스탠드얼론 앱. 메신저/결제 생태계 연결 없음 |
| 7 | **교육 레이어 부족** | 정보 제공은 하지만 해석 방법 가이드 약함 |
| 8 | **한정된 데이터 소스** | 증권사 리서치 리포트, 신용/대차 데이터 등 부재 |

## 3. 토스증권 분석

### 3-1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 운영사 | 비바리퍼블리카 (토스) |
| 가입자 | 720만명 |
| MAU | 410만명 |
| 해외주식 점유율 | ~20% (업계 1위, 키움증권 역전) |
| 외화증권 수탁 수수료 | 4,494억원 (업계 1위) |
| 영업이익 | 4,458억원 (+199.5% YoY) |
| 거래시간 | 하루 23시간 50분 (프리/정규/애프터마켓) |
| 수수료 | 0.1% 상시, 10달러 이하 무료 |

### 3-2. 핵심 기능 맵

| 영역 | 기능 | 상세 |
|------|------|------|
| **AI** | AI 어닝콜 | 해외기업 어닝콜 실시간 한국어 번역+요약. 누적 이용자 **150만명** (9개월). 자체 개발 번역 모델 (4만건 어닝콜 학습). 챕터별 보기, AI 관전 포인트, PIP 멀티태스킹 |
| **AI** | AI 시그널 | 뉴스/공시를 AI로 분석해 시장 변동 원인 실시간 제공. 3가지 자체 AI: 뉴스 분류, 금융 특화 번역, 리즈닝(Reasoning) |
| **매매** | 소수점 거래 | 1,000원부터 해외주식 투자 가능 |
| **매매** | 24시간 거래 | 프리마켓~애프터마켓까지 23시간 50분 |
| **소셜** | 커뮤니티 | MAU 180만 / DAU 75만, 하루 게시글 20만건 |

### 3-3. AI 기술 스택

```
[자체 AI 모델 3종]
├── 뉴스 분류 AI: 뉴스 카테고리/관련 종목 자동 분류
├── 금융 특화 번역 AI: 4만건 어닝콜 데이터 학습, 금융용어/수치 정확도 강화
└── 리즈닝 AI: 뉴스 맥락 분석 → 시장 영향 해석

[인프라]
├── vLLM + Triton Inference Server
└── LLM 기반 데이터셋 구축 자동화

[조직]
└── AI 트라이브 신설 (PM + ML 엔지니어 + 금융 도메인 전문가)
```

### 3-4. 강점 요약

- **압도적 스케일**: MAU 410만, 해외주식 점유율 1위, 수수료 수익 1위
- **AI 킬러 서비스**: 어닝콜 150만 이용자로 검증된 AI 제품력
- **자체 AI 모델**: 금융 도메인에 특화된 번역/분석 모델 자체 개발
- **활성 커뮤니티**: MAU 180만의 강력한 소셜 레이어
- **UX 철학**: "한 화면 한 기능" — 모바일 최적화 직관적 인터페이스
- **풀스택 증권사**: 정보+매매+커뮤니티 원스톱 제공

### 3-5. 약점 / 빈 공간

| # | 약점 | 설명 |
|---|------|------|
| 1 | **대화형 AI 부재** | AI가 정보 "제공"은 하지만, 사용자가 질문하고 대화하는 인터페이스 없음 |
| 2 | **포트폴리오 맥락 AI 없음** | "이 뉴스가 내 보유종목에 미치는 영향" 같은 개인화 분석 미제공 |
| 3 | **투자 교육 AI 미개척** | 초보 투자자를 위한 AI 기반 학습/가이드 기능 없음 |
| 4 | **A사 생태계 없음** | 토스 생태계는 있지만, A메신저(국민 메신저) 연동 불가 |
| 5 | **브로커리지 편중** | 수익의 70~85%가 거래 수수료 → 시장 하락기 리스크 |
| 6 | **콘텐츠 포맷 한정** | 텍스트/영상 중심, 팟캐스트/오디오 브리핑 같은 포맷 부족 |

## 5. A증권사이 이길 수 있는 빈 공간

### 5-1. vs 스톡나우 — 공략 포인트

| 빈 공간 | A증권사의 우위 |
|---------|---------------------|
| 매매 연동 없음 | **"읽고 바로 사기"** — 정보 소비에서 매매까지 원스톱 여정. 뉴스/어닝콜 화면에서 바로 주문 가능 |
| 개인화 불가 | **포트폴리오 데이터 보유** — 증권사만 가능한 보유종목 기반 맞춤 브리핑/알림 |
| 월 16,900원 구독 | **정보 무료 제공 가능** — 거래 수수료 모델이므로 정보를 무료로 풀어 거래 유도 |
| 스탠드얼론 | **A사 생태계** — A메신저 알림, A사 연동으로 접근성 극대화 |
| 교보증권 MOU 수준 | **자체 증권사** — 직접 서비스 통합 가능, 파트너 의존 없음 |

### 5-2. vs 토스증권 — 공략 포인트

| 빈 공간 | A증권사의 우위 |
|---------|---------------------|
| 대화형 AI 없음 | **AI 투자 어시스턴트** — "테슬라 실적 어때?", "내 포트폴리오 리스크는?" 같은 자연어 대화 |
| 포트폴리오 맥락 AI 없음 | **"이 뉴스, 나한테 어떤 의미?"** — 뉴스/어닝콜을 내 보유종목 맥락에서 AI가 해석 |
| 투자 교육 부재 | **AI 튜터** — 초보 서학개미를 위한 맞춤형 학습 콘텐츠 |
| A메신저 연동 불가 | **국민 메신저 레버리지** — A메신저으로 실시간 알림, 친구와 종목 공유, 채팅방 내 AI 호출 |
| 오디오 콘텐츠 없음 | **AI 모닝브리핑** — 스톡나우 스타일 팟캐스트를 A증권사 내에서 제공 |

### 5-3. 공통 빈 공간 (두 서비스 모두 못하고 있는 것)

> 이 영역이 A증권사의 **가장 큰 차별화 기회**

| # | 빈 공간 | 구체적 기회 |
|---|---------|------------|
| 1 | **대화형 AI 투자 어시스턴트** | 자연어로 질문하면 AI가 실시간 데이터 + 어닝콜 + 뉴스를 종합해 답변. "엔비디아 가이던스 올해 전망 어때?" → AI가 최신 어닝콜/뉴스/애널리스트 의견 종합 답변 |
| 2 | **포트폴리오 맥락 기반 개인화** | "내 보유종목 중 실적 시즌에 주의할 종목은?" 같은 개인화된 인사이트. 증권사만 가능 |
| 3 | **투자자 페르소나 기반 선제적 인텔리전스** | 8가지 투자 특성(1~5점)으로 유저 프로필링 → 확증편향을 깨는 보완적 관점 제안. "테크주 비중이 높아 FOMC 변동성 주의" 같은 맞춤 경고. 어떤 경쟁사도 투자 성향 기반 contrarian 제안을 하지 않음 |
| 4 | **구조화된 AI 기반 커뮤니티 (집단 지성)** | AI가 큐레이션한 공시/뉴스/이벤트가 게시글이 되고, 투자자들이 그 정보에 대한 의견을 댓글로 작성. 자유 게시판(토스증권)과 달리 특정 정보에 대한 집중 토론 → 더 깊은 논의, 높은 DAU/리텐션 |
| 5 | **AI 투자 교육** | "PER이 뭐야?", "이 기업 밸류에이션 어떻게 봐?" 같은 학습형 대화. 초보 서학개미 온보딩 |
| 6 | **메신저 기반 정보 전달** | A메신저 알림으로 어닝콜 요약, 포트폴리오 이슈 푸시. 앱을 열지 않아도 정보 도달 |

### 5-4. A증권사만의 고유 자산

경쟁사가 따라할 수 **없는** 구조적 우위:

```
┌──────────────────────────────────────────────────┐
│            A사 생태계 시너지                      │
│                                                    │
│  A메신저 ──→ AI 알림/브리핑 푸시                    │
```

## 6. 제안서 활용 시사점

### 스토리라인

1. **시장 기회 검증**: 스톡나우 MAU 3만(4개월), 토스증권 AI 어닝콜 150만 이용자 → 서학개미의 정보 접근성 문제는 **실재하는 대규모 시장 기회**
2. **기존 솔루션의 한계**:
   - 스톡나우: 정보는 좋지만 매매 연결 끊김, 개인화 불가
   - 토스증권: AI 기능 강하지만 대화형 AI/개인화 맥락 분석은 아직 미개척
3. **A증권사의 차별화 가설**:
   > "AI가 **내 포트폴리오 맥락**에서 미국 주식 정보를 해석해주고, **A메신저**을 통해 적시에 전달하면, 정보 소비 후 **매매 전환율**이 유의미하게 증가한다"

### 검증 지표 (안)

| 지표 | 측정 방법 |
|------|----------|
| AI 콘텐츠 → 매매 전환율 | AI 브리핑/알림 조회 후 24시간 내 주문 비율 |
| A메신저 알림 CTR | 카톡 알림 → 앱 유입 클릭률 |
| AI 대화 세션당 질문 수 | 사용자가 AI와 평균 몇 번 대화하는지 |
| 해외주식 거래 활성화 | AI 기능 사용자 vs 비사용자 월간 거래 건수 비교 |
| 신규 유입 | AI 기능 출시 전후 해외주식 계좌 개설 수 |
