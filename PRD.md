# 제품 요구사항 정의서 (PRD)
# AI 미국주식 브리핑 서비스

> **최종 업데이트:** 2026-03-21

## 1. 프로젝트 개요

**프로젝트명:** AI 미국주식 브리핑 서비스
**설명:** 한국 투자자의 미국 주식 정보 접근 격차를 AI/LLM으로 해소하는 서비스. SEC 공시·어닝콜·뉴스를 실시간 수집·분석하여 사용자 포트폴리오 기반으로 선제적 인사이트를 제공.
**형태:** A사 앱 내 웹뷰(Add-on) 방식의 웹 서비스

## 2. 문제 정의

한국 투자자는 3가지 구조적 벽에 막혀 있다:

| 벽 | 설명 |
|----|------|
| **언어의 벽** | SEC 공시·어닝콜이 복잡한 금융 영어로 작성 |
| **시차의 벽** | 미국장(한국 23:30~06:00)에서 발생하는 이벤트를 수면 중 놓침 |
| **맥락의 벽** | "EPS 상회"가 내 포트폴리오에 어떤 의미인지 해석해주는 서비스 없음 |

## 3. 타겟 고객

| 세그먼트 | 프로필 | 핵심 Pain Point |
|---------|--------|----------------|
| **직장인 투자자** (30~40대) | 포트폴리오 $10K~$100K, 10~20종목 | 본업이 있어 모니터링 시간 부족. 밤사이 이벤트 대응 불가 |
| **성장형 초보투자자** (20~30대) | $1K~$10K, 1~2년차 | 커뮤니티/유튜브 의존, 허위정보 판별 불가 |

## 4. 핵심 기능

### 기능 1: AI 종목 브리핑 (MVP)
- 매일 오전 7시, 보유종목 기반 개인화 브리핑 푸시
- 밤사이 어닝/공시/뉴스 교차 분석 → 포트폴리오 관점 요약
- 종목 간 연쇄 영향 분석 (NVDA 가이던스 → AMD 영향)
- 선제적 제안: "오늘 밤 FOMC + 보유종목 실적발표 겹칩니다. 변동성 대비 손절라인 설정하시겠어요?"

### 기능 2: AI 공시 해석기
- 10-K/10-Q/8-K 자동 파싱 → 한국어 해석
- 핵심 지표 변동, 리스크 팩터 변화, 경영진 톤 분석
- 공시 제출 즉시 해석 전달
- 전분기/전년 대비 비교 자동 생성

### 기능 3: AI 투자 맥락 Q&A
- 한국어 대화형 종목 분석 (공시/어닝콜/뉴스 기반)
- 출처 인용 필수
- 커뮤니티 정보 검증 (팩트체크)
- 투자 조언 금지 가드레일

## 5. UX Flow

```
A사 앱
  └─ 해외주식 탭
       └─ "AI 투자비서" 배너
            └─ WebView 로드
                 ├─ [홈] 오늘의 AI 브리핑
                 ├─ [공시] AI 공시 해석기
                 ├─ [Q&A] AI 투자 맥락 Q&A
                 └─ [설정] 관심종목/알림 설정
```

## 6. 기술 아키텍처

```
[데이터 수집] SEC EDGAR / Finnhub / NewsAPI / 경제캘린더
      ↓
[데이터 처리] 청킹 → 임베딩 → Vector DB (Pinecone/ChromaDB)
      ↓
[AI 분석] Claude API + RAG → 포트폴리오 매칭 → 인사이트 생성
      ↓
[프론트엔드] Next.js + Tailwind (모바일 웹뷰 375px)
```
